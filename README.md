# no-vs-sql
Benchmark application. Compare productivity of Realm vs SQLite with varoius conditions.

To run JSON test you require to add 'catalog.json' file in assets folder with  'catalog' array with next structure : <br>
<i>{<br>"catalog":[<br>{"artist":"Stacey Kent",<br>"title":"One Note Samba (Samba de uma nota so)",<br>"year":"2013",<br>"duration":184,<br>"id":0}<br>]<br>}</i> <br>
Currently it has only one entry
